# ProjecTra
Student Project Management and Discovery Platform

ProjecTra bridges the gap between project creation and discovery, providing students with a professional platform to build their portfolios while enabling educators to track and encourage student innovation. The platform fosters a collaborative academic environment where knowledge sharing and project inspiration thrive.

# System Overview
This is a project tracking web application built with:
- Frontend: HTML pages with vanilla JavaScript for client-side logic, handling user interactions, API calls, and UI updates.
- Backend: Node.js with Express.js, providing REST APIs for authentication, profiles, projects, notifications, and comments.
- Database: MySQL, storing users, projects, notifications, comments, and likes.
- Authentication: JWT-based, with middleware for token verification.
- Features: User registration/login, profile management, project CRUD, search/filter, likes, comments, notifications.
